# AI Crop Disease Detection from Drone Imagery

This project detects crop diseases from drone images using deep learning.

## Features
- Multi-crop disease detection
- Explainable AI using Grad-CAM
- Disease treatment suggestions
- Streamlit web interface

## Tech Stack
- Python
- PyTorch
- OpenCV
- Streamlit

## How it Works
1. Upload crop image
2. AI model predicts disease
3. Grad-CAM highlights infected region
4. System suggests treatment

## Run Locally
pip install -r requirements.txt
streamlit run app.py
